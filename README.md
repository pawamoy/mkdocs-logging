Logging utilities for MkDocs plugins.

## Installation

With `pip`:
```bash
pip install mkdocs-logging
```

With [`pipx`](https://github.com/pipxproject/pipx):
```bash
python3.7 -m pip install --user pipx
pipx install mkdocs-logging
```
